    What is Solid

    Solid is a Web-based decentralization ecosystem where people are in control over their own data.

    Solid achieves this by giving everyone control over their own personal data pod. Applications are completely separate, and have to ask permission to access your data.

    Querying Solid pods with Comunica

    Since Solid and Comunica have a compatible technology stack, Comunica can be used to query over Solid data pods. The default Comunica SPARQL engine can directly be used to query over public Solid data pods.

    If you want to query over data pods that require authentication, you will have to plug in the Solid-specific HTTP actor. The easiest way to do this is by either making use of LDflex or GraphQL-LD that ship with this Solid-specific actor by default. Alternatively, you can plug this Solid-specific actor into your own Comunica instance.
