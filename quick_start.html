Quick Start

Execute SPARQL queries in your browser

The easiest way to get started with Comunica, is by executing a SPARQL query from your browser.

Install Comunica SPARQL

In order to install Comunica SPARQL onto your machine, run the following command:
Requires Node.js ≥ 8.0 

$ npm install -g @comunica/actor-init-sparql

After that, you'll have access to the comunica-sparql command. This will allow you to for example execute the following SPARQL query on the DBpedia TPF interface: 

$ comunica-sparql http://fragments.dbpedia.org/2015-10/en \
      "CONSTRUCT WHERE { ?s ?p ?o } LIMIT 100"

You can also install Comunica without the -g flag to add it as a dependency to any project, after which you can use it to query programmatically.

More information on this command can be found by executing comunica-sparql --help or by consulting the Comunica SPARQL README or the Comunica manual.

Setup a development environment

If you want to contribute to Comunica, or develop your own module, make sure to check the Comunica monorepo README.
