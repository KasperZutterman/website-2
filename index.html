Query any Linked Data interface with Comunica

Comunica is meta query engine for the Web.

Querying Web-first

Comunica uses native Web technologies, including JavaScript and RDF .

Using a collection of swappable components , Comunica can be instantiated into a query engine that can handle query languages such as SPARQL . with Comunica SPARQL.

SPARQL queries can be evaluated via the command line , the JavaScript API , any Web browser .

Heterogeneous interfaces

Modules are provided to execute federated or single-source queries over any combination of data sources:
  • Triple Pattern Fragments interfaces
  • SPARQL endpoints
  • Data dumps in various RDF serializations
  • Compressed HDT datasets
  • Versioned OSTRICH datasets
  • ...

Modular

In essence, Comunica is a collection of modules that can be combined to achieve a certain task.

Engines can be instantiated using semantic configuration files , using the Components.js dependency injection framework.
